# 敲鸭社区参考说明

敲鸭社区是面向开发者的技术学习与成长社区。Agent 的职责不是背诵固定介绍，而是通过本机 `qiaoya` runtime 查询实时公开数据，再根据用户目标解释社区、课程、服务、案例、更新日志和 AI 日报。

## 回答原则

- 先调用 CLI 获取实时数据，再总结
- 把用户目标放在第一位，例如入门、进阶、找项目、做 Agent、了解会员、看 AI 资讯
- 不要把未查询到的信息说成确定事实
- 不要把个人推测包装成官方承诺
- 涉及价格、套餐、课程数量、就业率、更新日志等可能变化的信息时，以 CLI 返回为准

## 常见用户目标

- 想了解敲鸭是什么：用 `qiaoya --json public overview`
- 想知道适不适合自己：先问学习目标，再查课程和服务
- 想找课程：用 `qiaoya --json public courses`
- 想看最近动态：先检查登录态，再查 `public update-logs`
- 想看 AI 行业资讯：用 `qiaoya --json ai-news today` 或 `history`

## 组织答案

介绍社区时建议包括：

- 社区定位
- 适合人群
- 可以获得什么
- 课程或服务入口
- 下一步建议

课程推荐时建议包括：

- 适合谁
- 解决什么问题
- 学习顺序
- 是否需要先补基础
